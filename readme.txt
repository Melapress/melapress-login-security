== Description ==

Help users use strong passwords and boost the security of your WordPress site. Configure Strong Password Policies on your WordPress websites.

== Installation ==

1. Upload the `wp-password-policy-manager` directory to the `/wp-content/plugins/` directory.
2. Activate the WPassword plugin from the 'Plugins' menu in WordPress.
3. Activate the license key.
